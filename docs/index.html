The Lab of Animal Data Sciences

Leveraging data sciences and technology to support precision livestock farming and animal breeding

MISSION

Research Focuses

Providing computational solutions to modern farming challenges that limit production efficiencies and sustaniable practices.

Phenomics

Developing computational framework to extract information from video and sensor data

Genomics

Integrating phenomics-derived traits into genomics selection for better farming efficiency

Software

The research results will be delivered through open-source software and friendly interfaces

Projects

Virtual Tag

An unsupervised approach to track pig positions using one top-view RGB camera

Forcasting Pig Body Weight

Using RGB-D sensor-based imaging system coupled with the Python image-processing pipeline to provide an effective approach to predict the body weight of nonrestrained pigs.

XSimV2

XSimV2 is a fast and user-friendly tool to simulate sequence data and complicated pedigree structures
